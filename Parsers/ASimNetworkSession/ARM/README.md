# NetworkSessions - Azure Sentinel Information Model

This template deploys the following:
* imNetworkSessions - Network Session events from all normalized network providers
* vimNwSCheckPoint
* vimDnsCiscoASA
* vimNwSM365Defender
* vimNwSMPaloAlto9
* vimNwSWindowsFirewall
* vimNwSWireData
* vimNwSZScaler
